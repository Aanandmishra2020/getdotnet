Repair the .NET Framework on your machine

Problem

You need the .NET Framework to run many apps and games on Windows. Your app did not start, either because it is not configured properly or because the .NET Framework is not correctly installed.

Solution

You can repair your .NET Framework install using the .NET Framework Repair Tool. If that resolves your issue, that's great. If it doesn't, you'll need to contact the app or game publisher to ask for help.
