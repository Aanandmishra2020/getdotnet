Installing the .NET Framework 4 on Windows XP

Problem

You need to install the .NET Framework 4 or a later .NET version to run an app or game on Windows XP.

Solution

You can download and install the .NET Framework 4 on Windows XP. In some cases, you will may need a later update of the .NET Framework. The .NET Framework 4.5 and later .NET versions are not supported on Windows XP.

You can contact Microsoft for help if you cannot get the correct version of the .NET Framework installed.
